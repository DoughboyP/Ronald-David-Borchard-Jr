# Ronald David Borchard Jr - Adventure Game

An interactive Python adventure game following the life story of Ronald David Borchard Jr from birth to present day.

## Chapters
- Chapter 1: Birth to Infancy (0-5 years)
- Chapter 2: Childhood (5-12 years)
- Chapter 3: Adolescence (12-18 years)
- Chapter 4: Young Adulthood (18-25 years)
- Chapter 5: Adulthood (25+ years)

## How to Play
```bash
python main.py
```

## Structure
Each chapter branch contains:
- `story.py` - Game logic and narrative
- `story.md` - Story documentation
- `characters.py` - Character definitions
- `events.py` - Story events and choices

Explore each branch to experience different life chapters!
